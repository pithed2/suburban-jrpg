# Suburban JRPG

A retro-inspired, 16-bit style JRPG where ordinary suburban homeowner problems are treated with mythic seriousness.

The first milestone is a tiny playable vertical slice:

- Walk around
- Talk to NPCs
- Enter the basement
- Fight enemies
- Defeat the Evil Heating Coil
- Save the game
- Roll credits

See [docs/foundation.md](docs/foundation.md) for the core concept and scope rules.

## Intended Stack

- Phaser 3
- TypeScript
- Vite
- Tiled
- JSON-driven content
- Cloudflare Pages, Workers, and D1 later
